sdEditor内置很多可选参数，请根据自身需要配置。
参数列表
参数 用途 默认值 可选值 示范
toolbar 工具栏模式 full full、admin、mini toolbar:'full'
mode 编辑器模式 html html、code mode:'html'
height 编辑器高度 300 auto height:'300'
value 初始值 value:'初始值'
upload 上传接口路径 upload:'/upload.php'
params 上传接口参数 params:{token:"1234"}
save 保存远程图片接口路径 save:'/save.php'
arrow 下拉小箭头 false true、false arrow:true
autofocus 是否自动焦点 false true、false autofocus:true
autosave 自动同步 true true、false autosave:true
pagebreak 分页符 _editor_page_ pagebreak:'_sdcms_editor_page_'
fontfamily 下拉字体 fontfamily:['宋体','微软雅黑']
fontsize 下拉字体大小 fontsize:['12','14']
color 下拉颜色选择 color:['FFFFFF','000000']
headline 下拉标题选择 headline:[{t:'正文',c:'p',s:16},{t:'标题1',c:'h1',s:28}]
bgColor 小工具背景颜色 bgColor:['FFFFFF','000000']
plug 自定义按钮 plug:{}
